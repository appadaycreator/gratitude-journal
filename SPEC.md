# 感謝日記・ポジティブ思考記録【無料】無料Webツール - 技術仕様書

## 概要

**サービス名**: Gratitude Journal
**バージョン**: 1.5.0
**更新日**: 2026-06-13
**URL**: https://appadaycreator.com/gratitude-journal/

毎日3つの感謝を記録して幸福感を高める習慣づくりアプリ。登録不要・完全無料でご利用いただけます。

## データ管理

- **ストレージ**: ブラウザ localStorage（外部API通信なし）
- **永続化**: ページ読み込み時に自動復元
- **クリア**: ブラウザのサイトデータ削除で初期化

## 技術スタック

- HTML5 / CSS3 / Vanilla JavaScript
- PWA対応（manifest.json / Service Worker）
- レスポンシブデザイン（モバイルファースト）

## v1.5.0 改善内容（2026-06-13）

### M7: 比較・シミュレーション機能
- **比較テーブル**: 最大3つの日付の記録を並べて表示。気分・感謝内容・ひとことを一覧で比較
- **シミュレーター**: 最新の記録をベースに気分や感謝内容を変更し、「もし〜だったら」の仮想シナリオを実行。ポジティブ気分への転換や感謝の深掘りをサポート
- **UIトグル**: 比較テーブルとシミュレーターの切り替えボタンで直感的に機能を選択可能

## 使い方

1. 記録したい項目を入力フォームに入力する
2. 「追加」または「記録」ボタンをクリックする
3. 記録一覧で過去のデータを確認する
4. グラフや集計で傾向を把握する
5. 継続的に記録して変化を追跡する

## よくある質問（FAQ）

**Q: 感謝日記・ポジティブ思考記録は無料で使えますか？**

はい、完全無料・登録不要でご利用いただけます。

**Q: 記録データはどこに保存されますか？**

ご使用のブラウザのローカルストレージに保存されます。他のデバイスとは共有されません。

**Q: ブラウザを閉じても記録は残りますか？**

はい、同じブラウザを使う限りデータは保持されます。

**Q: 記録を削除するにはどうすればいいですか？**

リセットボタン、またはブラウザのサイトデータをクリアすることで削除できます。

**Q: スマートフォンで利用できますか？**

はい、スマートフォン・タブレット・PCすべてでご利用いただけます。
